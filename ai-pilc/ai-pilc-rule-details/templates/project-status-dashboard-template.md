## Hub Contribution Behavior

When AI-PILC renders this dashboard:

```
1. DETECT management_framework/dashboards/DASHBOARDS.md (hub marker).
2. IF hub exists:
   → CREATE or REFRESH project-status-dashboard.md in dashboards/.
   → Update the AI-PILC row in DASHBOARDS.md "Contributing Dashboards" table.
3. IF no hub exists:
   → CREATE dashboards/ + DASHBOARDS.md + project-status-dashboard.md.
4. NEVER modify another package's dashboard file.
```
