## AI-PPM Agent Shortcuts

| Shortcut | Agent | What It Does |
|----------|-------|--------------|
| `PGA__` | portfolio-governance-agent | Run full portfolio governance assessment (16 checks across 4 categories: currency, completeness, decision quality, health monitoring) |

### Usage

Type the shortcut anywhere in a chat prompt to trigger the agent.

**Example:** "PGA__ — check if our portfolio governance is up to date"

### When to Run

- Monthly (at minimum) as part of portfolio hygiene
- Before quarterly strategic reviews
- After onboarding a new portfolio manager
- When you suspect governance has lapsed (stale data, missing decisions)
