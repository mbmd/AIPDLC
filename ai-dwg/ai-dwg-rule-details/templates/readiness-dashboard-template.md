## Hub Contribution Behavior

When AI-DWG generates this dashboard:

```
1. DETECT management_framework/dashboards/DASHBOARDS.md (hub marker).
2. IF hub exists:
   → CREATE readiness-dashboard.md in dashboards/.
   → Register the AI-DWG row in DASHBOARDS.md "Contributing Dashboards" table.
3. IF no hub exists:
   → CREATE dashboards/ + DASHBOARDS.md + readiness-dashboard.md.
4. NEVER modify another package's dashboard file.
5. On RECONCILIATION: overwrite readiness-dashboard.md with updated snapshot.
```
