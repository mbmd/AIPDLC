## Hub Contribution Behavior

When AI-TGE renders this dashboard:

```
1. DETECT management_framework/dashboards/DASHBOARDS.md (hub marker).
2. IF hub exists:
   → CREATE or REFRESH quality-dashboard.md in dashboards/.
   → Update the AI-TGE row in DASHBOARDS.md "Contributing Dashboards" table.
3. IF no hub exists:
   → CREATE dashboards/ + DASHBOARDS.md + quality-dashboard.md.
4. NEVER modify another package's dashboard file.
```
